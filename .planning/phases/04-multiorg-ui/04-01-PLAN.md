# Phase 4 — Multi-org UI Plan

**Goal:** Complete mobile UI for org switching, governance, messaging, and recovery

**Scope:**
- Org switcher + directory
- Join flows (mint MembershipNFT via paymaster)
- Per-org chat (XMTP groups)
- DM list + threads
- Proposals list + detail
- Vote casting via paymaster
- Recovery setup + initiation
- Real-time state sync with Phase 3 subgraph

**Estimate:** 3–4 weeks (per SPEC)
